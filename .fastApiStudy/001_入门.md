## 一、路由、装饰器、参数

### 1. 路由
1. 如何访问对应的函数与返回值呢？
  通过FastAPI的路由如, http://127.0.0.1:8000/ 执行的就是@app.get("/")下的root()函数
2. FastApi的路由定义基于python的装饰器模式

### 2.装饰器 
1. 装饰器就是请求这个路径时，执行装饰器内的函数并返回结果
2. 装饰器格式: ` @app.get("路径")`。其中app是fastapi实例，get是请求方式，"/"是路由路径。如 http://127.0.0.1:8000/ 执行的就是@app.get("/")下的root()函数
```
@app.get("/")
async def root():
    return {"message": "Hello World11"}


@app.get("/hello/{name}")
async def say_hello(name: str):
    return {"message": f"Hello {name}"}
    
```
3. @装饰器 只作用于它下面紧挨着的一个 def 或 class。
4. 多个装饰器连续写在同一个函数上，表示这些装饰器都作用于同一个函数。
5. 在 FastAPI 中，@app.get("/path") 会把下面的函数注册为该路径的处理函数。
6. 同名函数后定义会覆盖前面的名字，但已注册到 FastAPI 路由表里的函数对象不会受影响。
7. 不建议写多个同名接口函数，容易影响可读性和接口文档。
### 3. 参数
```
路径中有 {参数名}  -> 用 Path
问号后面 ?a=1&b=2 -> 用 Query
请求体 JSON 数据   -> 用 Body /  pydantic中的BaseModel模型
```

#### 3.1 路径参数-类型注解Path
```
//fastapi允许为参数声明额外的信息和校验，如限制输入参数的范围。
// 导入Path函数
from fastapi import FastAPI, Path, Query

@app.get("/author/{name}/{b}")
async def author(name: str = Path(..., description="The author's name", min_length=2, max_length=10), b: int = Path(..., gt=0, lt=100)):
    return {"message": f"Author {name}, the number is {b}"}

```

#### 3.2 查询参数

```
# 导入Query函数
from fastapi import FastAPI, Path, Query

@app.get("/divide/find2")
async def divide(a: int = Query(0, gt=0, lt=100), b: int = Query(10, gt=0, lt=100)):
    if b == 0:
        return {"error": "Division by zero is not allowed."}
    return {"result": a * b}
```

#### 3.3 请求体参数

##### 3.3.1 请求体参数- BaseModel
    用于创建、更新资源
```
# 导入BaseModel函数
from pydantic import BaseModel
# 定义类型
class User(BaseModel):
    username: str
    password: str
# 类型注解
@app.post("/register")
async def register(user: User):
    return user

// 等同于 curl -X 'POST' 'http://127.0.0.1:8000/register&#39;-H 'accept:application/json' -H 'Content-Type:application/json' -d '{ "username": "daqiu", "password": "123456"} '
```

##### 3.3.2 请求体参数-类型注解 Field
    作用：添加参数注解，如默认值、是否必填等等。相较于只使用BaseModel可对类型做更多限制
```
# 定义类型
class User(BaseModel):
    username: str = Field(default="张三", description="The username", min_length=2, max_length=100)
    password: str = Field(default="123456", description="The password", min_length=6, max_length=100)

# 类型注解
@app.post("/register")
async def register(user: User):
    return user

```

#### 3.4 响应体参数
    默认情况下，FastAPI会自动将路径操作函数返回的Python 对象(字典、列表、Pydantic 模型等)，经由jsonable_encoder 转换为JSON兼容格式，并包装为 JSONResponse返回。这省去了手动序列化的步骤，让开发者能更专注于业务逻辑。如果需要返回非 JSON数据(如HTML、文件流)，FastAPI提供了丰富的响应类型来返回不同数据
##### 3.4.1 响应体参数- JSONResponse (默认返回)
    正常写即可，默认返回的就是JSON
##### 3.4.2 响应体参数- HTMLResponse
```
# 导入 fastapi.response
from fastapi.responses import HTMLResponse

# HTMLResponse示例
@app.get("/books/{book_id}", response_class=HTMLResponse)
async def get_book(book_id: int):
    # 模拟从数据库中获取书籍信息
    book_info = {
        "id": book_id,
        "title": "跨时代的AI",
        "author": "不留名的X先生",
        "price": 20.0
    }
    html_content = f"""
    <html>
        <head>
            <title>Book Info</title>
        </head>
        <body>
            <h1>Book ID: {book_info['id']}</h1>
            <p>Title: {book_info['title']}</p>
            <p>Author: {book_info['author']}</p>
            <p>Price: ${book_info['price']:.2f}</p>
        </body>
    </html>
    """
    return HTMLResponse(content=html_content, status_code=200)
```
##### 3.4.3 响应体参数- FileResponse
```
from fastapi.responses import FileResponse
# FileResponse示例
@app.get("/download/{file_name}", response_class=FileResponse)
async def download_file(file_name: str):
    file_path = f"./images/{file_name}"
    return FileResponse(path=file_path, filename=file_name, media_type='application/octet-stream')
```

##### 3.4.4 响应体参数- 自定义响应数据格式 response_model 
```
# 导入BaseModel函数
from pydantic import BaseModel
class news(BaseModel):
    id: int
    title: str
    content: str

@app.get("/createNews/{id}", response_model=news)
async def create_news(id: int):
    return {
        "id": id,
        "title": f"News Title {id}",
        # 注意，如果不按news中的返回会报错，如你删除content字段，返回的json中没有content字段，就会报错
        "content": f"This is the content of news item {id}."
    }
```

#### 3.5 异常处理
对于客户端引发的错误(4xx，如资源未找到、认证失败)，应使用"fastapi.HTTPException"来中断正常处理流程并返回标准错误响应
```
# 导入函数
from fastapi import FastAPI, HTTPException
# httpexception示例
@app.get("/get_news/{id}")
async def get_news(id: int):
    if id < 1 or id > 100:
        raise HTTPException(status_code=404, detail="News item not found")
    return {"id": id, "title": f"News Title {id}", "content": f"This is the content of news item {id}."}
```
## 二、HTTP 请求参数类型

  在HTTP协议中，一个完整的请求由三部分组成：
1. 请求行：包含方法、URL、协议版本
2. 请求头：元数据信息(Content-Type、Authorization等）
3. 请求体：实际要发送的数据内容

### 1. 路径参数

- **位置**：URL 路径的一部分  
  示例：`/book/{id}`

- **作用**：指向唯一的、特定的资源

- **方法**：`GET`

---

### 2. 查询参数

- **位置**：URL `?` 之后  
  示例：`k1=v1&k2=v2`

- **作用**：对资源集合进行过滤、排序、分页等操作

- **方法**：`GET`

---

### 3. 请求体

- **位置**：HTTP 请求的消息体（`body`）中

- **作用**：创建、更新资源，携带大量数据，如 `JSON`

- **方法**：`POST`、`PUT` 等
