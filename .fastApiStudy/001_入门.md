## 一、路由、装饰器、参数

### 1. 路由
1. 如何访问对应的函数与返回值呢？
  通过FastAPI的路由如, http://127.0.0.1:8000/ 执行的就是@app.get("/")下的root()函数
2. FastApi的路由定义基于python的装饰器模式

### 2.装饰器 
1. 装饰器就是请求这个路径时，执行装饰器内的函数并返回结果
2. 装饰器格式: ` @app.get("路径")`。其中app是fastapi实例，get是请求方式，"/"是路由路径。如 http://127.0.0.1:8000/ 执行的就是@app.get("/")下的root()函数
```
@app.get("/")
async def root():
    return {"message": "Hello World11"}


@app.get("/hello/{name}")
async def say_hello(name: str):
    return {"message": f"Hello {name}"}
    
```

### 3. 参数
```
路径中有 {参数名}  -> 用 Path
问号后面 ?a=1&b=2 -> 用 Query
请求体 JSON 数据   -> 用 Body /  pydantic中的BaseModel模型
```

#### 3.1 路径参数-类型注解Path
```
//fastapi允许为参数声明额外的信息和校验，如限制输入参数的范围。
// 导入Path函数
from fastapi import FastAPI, Path, Query

@app.get("/author/{name}/{b}")
async def author(name: str = Path(..., description="The author's name", min_length=2, max_length=10), b: int = Path(..., gt=0, lt=100)):
    return {"message": f"Author {name}, the number is {b}"}

```

#### 3.2 查询参数

```
// 导入Query函数
from fastapi import FastAPI, Path, Query

@app.get("/divide/find2")
async def divide(a: int = Query(0, gt=0, lt=100), b: int = Query(10, gt=0, lt=100)):
    if b == 0:
        return {"error": "Division by zero is not allowed."}
    return {"result": a * b}
```

#### 3.3 请求体参数

```
// 导入BaseModel函数
from pydantic import BaseModel
// 定义类型
class User(BaseModel):
    username: str
    password: str
// 类型注解
@app.post("/register")
async def register(user: User):
    return user

// 等同于 curl -X 'POST' 'http://127.0.0.1:8000/register&#39;-H 'accept:application/json' -H 'Content-Type:application/json' -d '{ "username": "daqiu", "password": "123456"} '
```


## 二、HTTP 请求参数类型

  在HTTP协议中，一个完整的请求由三部分组成：
1. 请求行：包含方法、URL、协议版本
2. 请求头：元数据信息(Content-Type、Authorization等）
3. 请求体：实际要发送的数据内容

### 1. 路径参数

- **位置**：URL 路径的一部分  
  示例：`/book/{id}`

- **作用**：指向唯一的、特定的资源

- **方法**：`GET`

---

### 2. 查询参数

- **位置**：URL `?` 之后  
  示例：`k1=v1&k2=v2`

- **作用**：对资源集合进行过滤、排序、分页等操作

- **方法**：`GET`

---

### 3. 请求体

- **位置**：HTTP 请求的消息体（`body`）中

- **作用**：创建、更新资源，携带大量数据，如 `JSON`

- **方法**：`POST`、`PUT` 等
